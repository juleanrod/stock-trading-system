# Stock Trading System

A full-stack web application that simulates a real-time stock trading platform.

## Features

- **Real-time Stock Trading**: Buy and sell stocks with live price updates.
- **Portfolio Management**: Track holdings and profit/loss.
- **Market Simulation**: Automated price generation and market hours.
- **Admin Dashboard**: Create stocks and manage market configuration.

## Tech Stack

- **Backend**: Node.js, Express, PostgreSQL, Socket.io
- **Frontend**: Next.js, React, Tailwind CSS, TypeScript

## Getting Started

### Prerequisites

- Node.js (v18+)
- PostgreSQL (v14+)

### Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   cd stock-trading-system/backend && npm install
   cd ../frontend && npm install
   ```
3. Set up environment variables (see `.env.example` in backend and frontend).
4. Run the development servers:
   - Backend: `npm run dev`
   - Frontend: `npm run dev`
